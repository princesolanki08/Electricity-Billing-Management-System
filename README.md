# ⚡ Electricity Billing System

A Java-based Electricity Billing System designed to manage customer information, meter readings, bill generation, and payment tracking efficiently.

---

## 🚀 Features

- 👤 Customer Management (Add, Update, View)
- ⚡ Meter Reading Input
- 💰 Automatic Bill Generation
- 📊 Payment Tracking
- 🗂️ Database Integration for data storage

---

## 🛠️ Technologies Used

- Java
- JDBC
- MySQL
